# DevHire API

## Overview

DevHire API is a backend service for a developer hiring marketplace where companies can post jobs and hire developers.

## Features

* JWT Authentication
* Role-based authorization (Company / Developer / Admin)
* Job posting & applications
* Messaging system
* Reviews & ratings
* Contract management
* Caching for performance
* RESTful API design

## Tech Stack

* Node.js + Express
* MongoDB / PostgreSQL
* Redis (Caching)
* JWT Auth
* Docker (optional)

## Installation

```bash
git clone https://github.com/yourusername/devhire-api.git
cd devhire-api
npm install
npm run dev
```

## API Base URL

```
http://localhost:5000/api
```
